# UI Kit — AI Ops OS · LinkedIn Inbound Triage Agent

The operator surface for the first product in the AI Ops Consultant OS suite. The operator (you, the consultant) reviews leads the agent has triaged from LinkedIn inbound, approves drafted replies, and tracks pipeline ROI.

## Files

- `index.html` — interactive demo. Sidebar nav, lead queue, thread view with speech bubbles, approval flow.
- `Sidebar.jsx` — fixed left rail, nav, agent status pulse
- `LeadQueue.jsx` — list of leads with intent score chips
- `ThreadView.jsx` — speech-bubble conversation + agent reasoning + approve/revise actions
- `Composer.jsx` — drafted-reply review surface
- `MetricCard.jsx` — pipeline / ROI / response-time stat cards
- `TopBar.jsx` — search + filter + agent activity indicator

## Note

No codebase or Figma was attached. This UI is invented from the brand spec — a faithful application of the design system, not a copy of an existing product. When real screens or code arrive, this kit should be revised to match.
